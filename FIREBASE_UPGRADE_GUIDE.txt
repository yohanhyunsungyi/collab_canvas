STEP-BY-STEP UPGRADE INSTRUCTIONS:

STEP 1: Click This Link
──────────────────────────────────────────────────────────────────────
https://console.firebase.google.com/project/gauntlet-collabcanvas-7d9d7/usage/details

This opens your Firebase project's billing page.


STEP 2: Click "Upgrade" Button
──────────────────────────────────────────────────────────────────────
Look for:
- "Upgrade" button (top right)
- or "Modify plan" link
- or "Switch to Blaze" option


STEP 3: Select Blaze Plan
──────────────────────────────────────────────────────────────────────
You'll see plan options:
☐ Spark (Free) - Current plan
☑ Blaze (Pay as you go) - Select this!

Click "Continue" or "Select Plan"


STEP 4: Add Billing Account
──────────────────────────────────────────────────────────────────────
Option A: Use Existing Google Billing Account
  - Select from dropdown if you have one

Option B: Create New Billing Account
  - Click "Create billing account"
  - Enter credit card information
  - Add billing address
  - Click "Submit and enable billing"


STEP 5: Set Budget Alerts (HIGHLY RECOMMENDED!)
──────────────────────────────────────────────────────────────────────
After upgrading:
a) Go to: https://console.cloud.google.com/billing
b) Click your billing account
c) Click "Budgets & alerts" (left menu)
d) Click "CREATE BUDGET"
e) Set budget: $10/month
   Alert at: 50%, 90%, 100%
f) Add notification email

This ensures you get warned before any charges!


STEP 6: Confirm Upgrade
──────────────────────────────────────────────────────────────────────
You should see:
✓ "Blaze" badge on your Firebase project
✓ "Cloud Functions" menu item enabled
✓ No more upgrade warnings


STEP 7: Deploy Cloud Functions
──────────────────────────────────────────────────────────────────────
Now run in your terminal:

cd /Users/yohanyi/Desktop/GauntletAI/01_CollabCanvas
firebase deploy --only functions

This will deploy the secure Cloud Functions!

WHAT IF I DON'T WANT TO UPGRADE?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  WARNING: Your API key will remain EXPOSED!

Without Cloud Functions, you have these options:

Option 1: Disable AI Features (Safest)
  - Comment out AI components
  - Remove AI from the app
  - Deploy without AI

Option 2: Use Client-Side (NOT RECOMMENDED - INSECURE!)
  - Keep using the old code
  - API key remains exposed
  - Anyone can steal and use your key
  - Could cost you money!

I STRONGLY recommend upgrading. The Blaze plan is essentially free
for your usage level and provides proper security.


AFTER UPGRADE - COMPLETE SECURITY STEPS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Once upgraded, follow these steps from SECURITY_FIX_SUMMARY.txt:

1. Revoke old API key (https://platform.openai.com/api-keys)
2. Create new API key
3. Install functions dependencies (cd functions && npm install)
4. Configure Firebase (firebase functions:config:set openai.key="NEW_KEY")
5. Deploy functions (firebase deploy --only functions)
6. Rebuild & deploy (npm run build && firebase deploy --only hosting)
7. Verify security (grep -r "sk-" dist/)
